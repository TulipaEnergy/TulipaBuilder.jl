TulipaBuilder

TulipaBuilder aims to make it easy to create a Tulipa problem. It provides a graph-based constructive interface to defining the assets and flows of an energy system, their respective data, and profiles. The interface is heavily tied to TulipaEnergyModel's input schema, but it glosses over the complexity.

Warning

TulipaBuilder is still in development, and there is no guarantee that it is creating the model that you expect. Please check the output and open issues if you see something wrong.
